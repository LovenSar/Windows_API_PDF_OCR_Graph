WINHTTP_SECURITY_INFO 结构
(winhttp.h) 项目2023/08/27
WINHTTP_SECURITY_INFO 结构包含请求的 SChannel 连接和密码信息。
语法 C++
typedef struct _WINHTTP_SECURITY_INFO {
  SecPkgContext_ConnectionInfo ConnectionInfo;   SecPkgContext_CipherInfo     CipherInfo;
} WINHTTP_SECURITY_INFO, *PWINHTTP_SECURITY_INFO; 成员
ConnectionInfo
包含 请求的 SChannel 连接信息的SecPkgContext_ConnectionInfo。 CipherInfo
包含 请求的 SChannel 密码信息的SecPkgContext_CipherInfo。注解
此结构与 WinHttpQueryOption 一起使用，通过指定
WINHTTP_OPTION_SECURITY_INFO 标志来检索请求的安全信息。要求
 
最低受支持的客户端最低受支持的服务器标头
